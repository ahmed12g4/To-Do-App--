# ✅ To-Do App

تطبيق بسيط لإدارة المهام اليومية.  
تقدر تضيف وتحذف المهام، وبيتم حفظها تلقائيًا باستخدام LocalStorage.

## 🛠 المميزات

- إضافة مهمة جديدة
- حذف مهمة من القائمة
- حفظ تلقائي للمهام حتى بعد ما تقفل الصفحة
- عرض التاريخ والوقت لكل مهمة

## 💻 التقنيات المستخدمة

- HTML
- CSS
- JavaScript (vanilla)
- LocalStorage
